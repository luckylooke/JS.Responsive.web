Module: documentState

Detection of document loading state.
Classes applied to <HTML>:
  • state-uninitialized - has not started loading yet
  • state-loading - is loading
  • state-interactive - Has loaded enough and the user can interact with it
  • state-complete - fully loaded
  • state-loaded - when document is loaded (including all images)
  • state-unloading - when document is unloading
Events:
  • changedDocumentState - Arguments: {String} - new-state, {String} - old-state,event: both are states strings like classes but without "state-" prefix
