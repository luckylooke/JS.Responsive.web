Module: detectHiRes

Detection of high resolution display
Classes applied to <HTML>:
  • hires-display - applied when display is high resolution (DEMO)
  • normal-display - applied when display is lower resolution (DEMO)
  • display-pixel-ratio-$$-more - current display pixel ratio is higher then $$ value
  • display-pixel-ratio-$$-less - current display pixel ratio is lower or equal then $$ value

Example usage of module
