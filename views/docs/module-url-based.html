<static> JS.Responsive.urlChanged

@(

Method for use cases of SPA routers which dynamically changes url via js. If your router uses # before navigation urls, like mypage.com/#/my/router/url. Then you don't need to use this method, as library is listening on window hashchange event. And generates hash- classes accordingly.

)
@(Since):
  • @(3.2.0)

@(Source:)
  • url-based.js, line 27
@(Example)
